## Requesting access

Codespaces access is managed in [ministryofjustice/octo-access](https://github.com/ministryofjustice/octo-access). To request it, open the **GitHub Codespaces Access Request** issue form there, or read [Prerequisites and access](docs/prerequisites-and-access.md).

## Format

The pages are written in GOV.UK Tech Docs style (Markdown with front matter) so they can be published with the Tech Docs Template later. They also render directly on GitHub in the meantime.
